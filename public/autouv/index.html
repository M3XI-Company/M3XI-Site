What's new

Release notes

What’s new in v1.1.

Rolling out now — the app checks for it on launch, or grab it any time from Help → Check for updates.

  • No more phantom cube. Importing FBX/OBJ/glTF no longer drags Blender’s default cube into your unwrap or export.
  • A UV map per part. Every separated part now gets its own tightly-packed UV layout instead of sharing one crowded atlas.
  • Space actually used. Islands scale and nest to fill the 0–1 space — no more big empty margins.
  • Fewer, cleaner seams. Hand-painter-friendly cuts: seams land only on real hard edges, not every little crease.
  • Everything in one place. All unwrapped models are collected in a single AutoUV_Unwraps folder next to your file.
  • Sharper taskbar icon and a built-in update checker.
